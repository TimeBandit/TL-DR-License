# TL;DR License

Helps you find a license for your project.

## Running application locally

1. Clone the project.
1. Run `npm install` in __tldrlicense__.
1. Run `gulp serve` in __tldrlicense__ to make a build.
1. Go to [localhost:3000](http://localhost:3000/)
